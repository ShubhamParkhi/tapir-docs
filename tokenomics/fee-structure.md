# Fee Structure

| Fee Type                | Rate | Distribution                              |
| ----------------------- | ---- | ----------------------------------------- |
| **Depeg AMM Swap Fees** | 0.3% | 50% to LPs, 50% to `TAPIR` stakers.       |
| **Fixed Yield Fees**    | 0.2% | 70% to protocol treasury, 30% burned.     |
| **Withdrawal Fees**     | 0.1% | Entirely burned to reduce `TAPIR` supply. |
